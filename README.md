A high-fidelity frontend clone of the **UltraEdit** software download page. This project was developed as a practice exercise to master advanced CSS layouts, responsive design, and semantic HTML structure without the use of external frameworks.

**🚀 Live Demo:** [View the live site on Vercel](https://ultra-edit-website-clone-html-au7evh2jw-malaikaahsans-projects.vercel.app/)

---

## 🚀 Features

* **Responsive Grid System:** Uses CSS Grid to transition smoothly from a 3-column desktop layout to a single-column mobile view.
* **Semantic HTML5:** Built using meaningful tags like `<header>`, `<nav>`, `<main>`, and `<footer>` for better accessibility and SEO.
* **Custom UI Components:** Includes styled download buttons, hover effects, and a custom navigation bar.
* **Professional Footer:** A multi-column, dark-themed footer that stays fully responsive across all devices.

## 🛠️ Tech Stack

* **Markup:** HTML5
* **Styling:** Vanilla CSS3
* **Deployment:** Vercel
